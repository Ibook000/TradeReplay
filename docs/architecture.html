TRADEREPLAY ARCHITECTURE

Multi-Exchange Contract Trade Replay System

System Architecture

Frontend

  • - Vanilla JS SPA, no framework
  • - LightweightCharts (TradingView)
  • - Chart.js for statistics
  • - Responsive dark theme

Backend

  • - FastAPI + Uvicorn
  • - Async HTTP (httpx)
  • - PostgreSQL (psycopg2)
  • - Background scheduler

Data Sources

  • - OKX: trade + K-line data
  • - Bybit: trade + K-line data
  • - Binance: K-line fallback
  • - DeepSeek: AI analysis
